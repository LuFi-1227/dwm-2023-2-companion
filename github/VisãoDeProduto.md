# Declaração de visão de produto:
```
Para alunos de ciência da computação 
Que não conseguem ter acesso às informações e contatos importantes para o andamento do curso
O Companion é um repositório de dados institucionais do curso de Ciência da Computação 
Que te fornecerá os horários de aula, calendários com eventos, estatutos, links e manuais referentes ao curso,
para você não se perder nem se prejudicar no andamento do seu curso, pois
Ao contrário do BENDER
O nosso produto está com dados institucionais atualizados.
```
